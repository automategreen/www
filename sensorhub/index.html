Introducing the Sensor Hub!

The sensor Hub makes wired sensors easy. There are a lot of cheap wired sensors, but they are hard to use. Sensor hub solves this problem by making it as easy as plugging in a wire. The Hub provides support simultaneous use of six sensors and three relays on a single hub. We based it on Particle's excellent IoT platform, which provides support for either WiFi or 3G mobile connectivity.
